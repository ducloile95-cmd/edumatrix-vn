# workers/messenger/.dev.vars
META_PAGE_ACCESS_TOKEN=<page-access-token>
META_APP_SECRET=<meta-app-secret>
META_WEBHOOK_VERIFY_TOKEN=<random-verify-token>
FIREBASE_CLIENT_EMAIL=<service-account-email>
FIREBASE_PRIVATE_KEY="-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n"
npx wrangler login
npx wrangler secret put META_PAGE_ACCESS_TOKEN --env production
npx wrangler secret put META_APP_SECRET --env production
npx wrangler secret put META_WEBHOOK_VERIFY_TOKEN --env production
npx wrangler secret put FIREBASE_CLIENT_EMAIL --env production
npx wrangler secret put FIREBASE_PRIVATE_KEY --env production
npm run deploy:prod

Xử lý lỗi theo triệu chứng

Chỉ thay một cấu hình mỗi lần, sau đó kiểm thử lại đúng lớp vừa thay.

Triệu chứngNguyên nhân thường gặpCách xử lý
Meta không verify webhookCallback sai path hoặc verify token không trùngDùng HTTPS /webhook; đặt lại cùng một token trong Meta và Worker production.
Worker trả 401Thiếu hoặc hết hạn Firebase ID tokenĐăng nhập lại Edumatrix; kiểm tra Firebase project ID và service-account secret.
Worker trả 403 admin_requiredTài khoản không phải AdminKhông nới quyền Worker; dùng đúng tài khoản Admin cho đăng Fanpage.
Facebook từ chối gửiPage token, permission, App mode hoặc cửa sổ MessengerXem Worker logs, kiểm tra token/Page, App Review và chính sách gửi hiện hành.
CORS khi gọi WorkerALLOWED_ORIGIN không khớp frontendĐặt đúng scheme + hostname + port, không có path; deploy lại đúng environment.
Google báo origin mismatchAuthorized JavaScript origins thiếu hoặc có pathThêm chính xác origin local/production; chờ cấu hình cập nhật rồi thử lại.
API key bị chặnHTTP referrer hoặc API restriction saiKiểm tra referrer có /*, Picker API được bật và key thuộc đúng Cloud Project.

Điều kiện trước khi go-live

Chỉ công bố khi kiểm thử bằng tài khoản và Page production đã hoàn tất.

FacebookApp ở trạng thái phù hợp, webhook delivery ổn định, quyền đã được duyệt và token chưa hết hiệu lực.
WorkerNăm secret có trong production, origin đúng, health 200 và log không chứa dữ liệu bí mật.
GoogleConsent screen, origins, test users/publishing status và API key restrictions đã đúng domain.
Ứng dụngBuild production mới nhất, Admin settings đúng, smoke test Chat/Fanpage/Drive đều đạt.
Checklist được lưu trong localStorage của trình duyệt này. Không có secret nào được tài liệu lưu lại.

Nguồn chính thức và mã nguồn đối chiếu

Tên biến và giới hạn lấy từ mã nguồn dự án; thao tác nền tảng được đối chiếu với tài liệu nhà cung cấp.

  • Cloudflare Worker SecretsCách lưu và triển khai secret an toàn.

  • Meta Messenger PlatformThiết lập Messenger, token và chính sách nền tảng.
  • Meta Graph API WebhooksCallback verification và webhook subscription.
  • Google OAuth Token ModelAccess token ngắn hạn cho ứng dụng trình duyệt.
  • Google Picker for WebClient ID, API key, Project Number và PickerBuilder.
  • Google Drive ScopesLựa chọn và đặc tính của scope drive.file.
  • Google Drive UploadsMultipart upload và các phương thức tải tệp.
