Kết nối Facebook API

Hoàn thành tuần tự từ Meta App, Page, quyền, Worker secret đến App Review.

Tạo Meta App

Mở Meta for Developers, tạo app thuộc doanh nghiệp và thêm sản phẩm Messenger. Điền App Domain, Privacy Policy và User Data Deletion URL.

Kết nối Facebook Page

Trong Messenger Settings, kết nối đúng Page của trung tâm. Tài khoản thao tác phải có toàn quyền trên Page.

Cấp quyền phù hợp

Chuẩn bị pages_messaging, pages_manage_posts, pages_read_engagement, pages_manage_metadata và pages_show_list theo đúng luồng sử dụng.

Lấy token và App Secret

Kiểm tra token bằng Access Token Debugger. Không sao chép token vào Cài đặt của Edumatrix hoặc tệp env frontend.

Nhập Cloudflare Secret

Dùng đúng environment production để tránh nhập secret vào Worker local.

Set-Location workers/messenger
npx wrangler login
npx wrangler secret put META_PAGE_ACCESS_TOKEN --env production
npx wrangler secret put META_APP_SECRET --env production
npx wrangler secret put META_WEBHOOK_VERIFY_TOKEN --env production
npx wrangler secret put FIREBASE_CLIENT_EMAIL --env production
npx wrangler secret put FIREBASE_PRIVATE_KEY --env production

Build và deploy

Set-Location workers/messenger
npm install
npm test
npm run build:prod
npm run deploy:prod

Cấu hình frontend

VITE_MESSENGER_WORKER_URL=https://<worker-domain>
Graph API version: dự án đang dùng v22.0. Chỉ nâng phiên bản sau khi đọc changelog và kiểm tra lại send, post và webhook.

Thiết lập Webhook HTTPS

Cloudflare Worker cung cấp HTTPS. Callback của Meta phải trỏ trực tiếp tới endpoint webhook.

Nhập Callback URL

Dùng https://<worker-domain>/webhook trong Messenger Settings > Webhooks.

Nhập Verify Token

Giá trị phải giống META_WEBHOOK_VERIFY_TOKEN. Không dùng App Secret làm Verify Token.

Chọn sự kiện

Subscribe messages, messaging_postbacks và messaging_referrals, sau đó đăng ký đúng Page.

Kiểm tra chữ ký POST

Worker dùng raw body và X-Hub-Signature-256. Payload sai chữ ký phải nhận HTTP 401.

curl.exe "https://<worker-domain>/webhook?hub.mode=subscribe&hub.verify_token=<VERIFY_TOKEN>&hub.challenge=edumatrix-ok"

Liên kết phụ huynh

https://m.me/<PAGE_USERNAME>?ref=<FIREBASE_UID_PHU_HUYNH>

Khi phụ huynh gửi tin, Worker ghi messenger_connections, chat_threads và message con. Module Chat chỉ hiển thị cho Admin hoặc giáo viên được phân công.

Kết nối Google Drive

Folder ID hiện chỉ là cấu hình công khai. Cần OAuth thật trước khi Edumatrix có thể tải tệp.

Phương án phù hợp Firebase Spark: Google Identity Services token model ở frontend, scope drive.file, access token chỉ giữ trong bộ nhớ.

Enable API

Trong Google Cloud Console, enable Google Drive API. Enable thêm Google Picker API nếu dùng giao diện chọn thư mục.

Cấu hình consent screen

Điền tên app, email hỗ trợ, domain, Privacy Policy và audience. Khi Testing, thêm tài khoản Admin vào Test users.

Tạo OAuth Web Client

Authorized JavaScript origins gồm http://localhost:5173, domain Firebase Hosting và custom domain nếu có. Origin không chứa path.

Xin scope tối thiểu

Dùng https://www.googleapis.com/auth/drive.file. Không xin quyền toàn bộ Drive nếu không cần.

Chọn hoặc tạo thư mục

Dùng Google Picker hoặc tạo thư mục qua API sau consent. Sau đó mới lưu Folder ID vào Cài đặt.

Upload đúng loại

Dùng multipart cho tệp nhỏ tối đa 5 MB. Dùng resumable upload cho tệp lớn hoặc mạng không ổn định.

Biến frontend dự kiến

VITE_GOOGLE_DRIVE_CLIENT_ID=<oauth-web-client-id>
VITE_GOOGLE_API_KEY=<api-key-neu-dung-picker>
VITE_GOOGLE_CLOUD_PROJECT_NUMBER=<project-number-neu-dung-picker>
Các biến Drive chưa được mã nguồn hiện tại sử dụng. Cần triển khai service OAuth, Picker và upload trước khi thêm vào production.

Quy tắc thư mục

Folder ID lấy từ https://drive.google.com/drive/folders/<FOLDER_ID>. Dán ID thủ công không tự cấp quyền. Thư mục phải được chọn bằng Picker hoặc do Edumatrix tạo sau khi người dùng cấp quyền.

Xử lý lỗi thường gặp

Mở từng mục để xem nguyên nhân và cách khắc phục.

Meta không Verify and Save webhook

Kiểm tra URL public HTTPS, Verify Token đúng environment và Worker đã deploy. Gọi thử handshake bằng curl.

Webhook GET thành công nhưng POST trả 401

App Secret không khớp hoặc raw body đã bị thay đổi trước khi tính HMAC. Đặt lại META_APP_SECRET và giữ raw request body.

Chat hiển thị trạng thái chỉ đọc

Frontend thiếu VITE_MESSENGER_WORKER_URL. Thêm biến đúng Worker domain rồi build lại.

Worker trả no_recipient

Phụ huynh chưa liên kết Page. Gửi link m.me?ref=<uid> và yêu cầu họ gửi một tin nhắn.

Google báo origin_mismatch

Authorized JavaScript origin phải khớp scheme, host và port. Không thêm path hoặc dấu gạch cuối không cần thiết.

Drive trả 404 cho Folder ID

Access token chưa có quyền trên thư mục. Chọn lại bằng Picker hoặc tạo thư mục qua API sau consent.

Drive upload trả 401

Access token đã hết hạn. Yêu cầu Admin bấm kết nối lại. Không cố giữ token bằng Local Storage.
